### tips

- 在线OJ使用[牛客网《剑指Offer》](https://www.nowcoder.com/ta/coding-interviews?page=1)
- 题目选择牛客网提供完善测试用例的题目
- 一个题目一个目录，目录中README及题目。若题目复杂，直接粘贴牛客网地址
- 每个答题人，在题目目录中建立自己的目录。将题目的答案用自己熟悉的方式实现，在牛客网验证通过后，并将答案提交至此目录
- 本项目用于帮助大家记录、回溯

```
----topic 001
--------your_name_person1
------------answer.cpp
------------my_readme.md
--------your_name_person2
```
